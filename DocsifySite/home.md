## Overview

The 3D Platformer Template is a game-ready character controller for Unreal Engine built around one thing: **how it feels in your hands.**

It provides a complete platformer foundation with responsive controls, procedural animations, particle effects, and sound design — all without requiring a humanoid character rig or pre-made animations. Drop it into your project and start building levels.

## What's Included

- **Core Movement** — Run, jump, and double jump with tight, weighted controls
- **Procedural Animations** — Forward tilt, directional lean, squash and stretch driven in real time
- **Particle Effects** — Jump burst, land impact, and walk dust — wired up automatically
- **Sound Effects** — Punchy jump/land SFX and rhythmic footsteps synced to speed
- **Bean Character** — Simple capsule mesh that feels alive through procedural motion
- **Full Configuration** — Every parameter exposed in the Blueprint Details panel
- **Example Level** — Working demo level showing everything in action

## Requirements

| Requirement | Details |
|---|---|
| Engine Version | Unreal Engine 5.6+ |
| Plugins | None required |
| Project Type | Blueprint-only or C++ |
| Platform | Windows (Mac/Linux untested) |
| Input | Keyboard, Mouse, Gamepad |
| Multiplayer | Not supported (single-player only) |
